## 🎯 How to Use

### For Complete Beginners
Start from File 1 and go in order.

### For Quick Interview Prep
1. Read `ml-fundamentals.md` (get the big picture)
2. Skim `supervised-learning.md` (know the algorithms)
3. Read `deep-learning-basics.md` (CNN, RNN, Transformers)
4. Finish with `ml-interview-questions.md` (rapid-fire revision)

### For NLP Roles
Focus on: `nlp-basics.md` + `deep-learning-basics.md` (Transformers section)

- ❌ No heavy math or formulas
- ❌ No code implementations (this is conceptual understanding)
